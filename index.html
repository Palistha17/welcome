Assignment on GIT and VCS

1. Can we have a global .gitignore?

Yes, we can create a global .gitignore file and all the entries in that file will be ignored in all the Git repositories. For this we use the following command:
git config –global core.excludesfile

2. How to find difference between two branches?

To find the difference between two branches, we use “git diff” command provided that the branch names are separated by dots.
git diff branch1..branch2

3. How to find change of a file between two commits?

Using the following command, we can find change of a file between two commits:
git diff (commit1-hash) (commit2-hash) -- (filename)
